Los tres pilares de la Web

Tim Berners-Lee conocido por ser el padre de la World Wide Web:

  • HTML
  • HTTP
  • URL

HTML

HTML, acrónimo en inglés de HyperText Markup Language («lenguaje de marcado de hipertexto»), hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

Fuente: HTML, wikipedia

HTTP

El protocolo de transferencia de hipertexto (en inglés: Hypertext Transfer Protocol, abreviado HTTP) es el protocolo de comunicación que permite las transferencias de información a través de archivos (XML, HTML…) en la World Wide Web.

Fuente: HTTP, wikipedia

URL

Un LRU o localizador de recursos uniforme ( conocido por las siglas URL, del inglés Uniform Resource Locator)​ es un identificador de recursos uniforme (Uniform Resource Identifier, URI) cuyos recursos referidos pueden cambiar, esto es, la dirección puede apuntar a recursos variables en el tiempo.

Fuente: URL, wikipedia
